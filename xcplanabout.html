About XCWebPlan

© 2016 Richard Brisbourne

XCWebPlan is an open source program written almost entirely in Javascript (with a bit of php to interrogate the airspace database). It is based on Google Maps and makes use of their javascript apl.

Program code

The Javascript code for the application itself was mostly written by Richard Brisbourne, with help and support from Alistair Green. Apart interacting Google Maps api, it also uses some other libraries and plugins:

  • JQuery- a javascript library well-known to web developers
  • Infobox- from the Google Maps utility library
  • DCalendar- JQuery plug-in used for picking dates

Data

Maps and satellite images are from Google Maps.

Airspace data is drawn from a number of places, the largest being soaringweb.org. Unfortunately very little is available in computer-readable format from official sources so we are dependant on people who work voluntarily to create the files we all use in our software, both on the ground and in our gliders, and who deserve more credit than they often get. The data is held in tables on the server, so has been arrived at by conversion of some of these files.
