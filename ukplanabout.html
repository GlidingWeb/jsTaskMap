About XCWebPlan (UK version)

© 2016 Richard Brisbourne

XCWebPlan is an open source program written almost entirely in Javascript (with a bit of php to interrogate the airspace database). It is based on Google Maps and makes use of their javascript apl.

Credits

Program code

The Javascript code for the application itself is open source and was mostly written by Richard Brisbourne, with some help and support from Alistair M Green. Copyright is held Richard Brisbourne; for further information for how to access the source code see below, for information on how you are entitled to make use of it see the licence.

Apart interacting Google Maps api, it also uses some other libraries and plugins:

  • JQuery- a javascript library well-known to web developers
  • Infobox- from the Google Maps utility library
  • DCalendar- JQuery plug-in used for picking dates

Data

Maps and satellite images are from Google Maps.

Airspace data is held in tables on the server, generated from data sourced using Geoff Brown's AS Select program. Any responsibility for conversion errors or failure to remain in step with Geoff's data is mine alone.

Turning point information is extracted from the Excel file at http://www.spsys.demon.co.uk/turningpoints.htm.
